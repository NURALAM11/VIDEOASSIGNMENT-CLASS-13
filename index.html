CSS Before & After Selector

What is CSS ::before Selector ?

The CSS ::before selector inserts content before a selected element.These selectors are commonly used to add text before or after a paragraph or a link.

When you’re designing a website, we may want to add in content before the content of an element appears. For instance, you may want to add an image before each bullet point in a list.

That’s where the CSS ::before pseudo-elements come in. This pseudo-elements allow us to insert content before the content of an element.

A pseudo-element is a special keyword that allows you to style a specific part of the elements selected by a CSS selector.

The syntax of the ::before pseudo-element is

selector::before { // CSS rules }

The ::before selector is added after the name of the element to which you want to apply the selector.

Here are the main components of the ::before pseudo-element:

selector refers to the selector to which ::before should be applied. So, if you want to apply a ::before element to every “a” tag, your selector would be “a”. Or if you wanted to apply a ::before element before every element with the class name “link”, you would use the “.link” selector.

::before tells the browser to add the content of the CSS rules before the selector.

CSS rules are the rules that should be applied before the selector.

What is CSS ::after Selector ?

The CSS CSS :after inserts content after a specified element.These selectors are commonly used to add text before or after a paragraph or a link.

When you’re designing a website, we may want to add in content after the content of an element appears. For instance, we may want to add an image after each bullet point in a list.

That’s where the CSS ::after pseudo-elements come in. This pseudo-elements allow you to insert content after the content of an element.

A pseudo-element is a special keyword that allows we to style a specific part of the elements selected by a CSS selector. We’re going to refer to pseudo-elements as selectors throughout this article because every pseudo-element is a selector.

The syntax of the ::after pseudo-element is as follows:

selector::after { // CSS rules }

The syntax for ::after is the same as the syntax for the ::before pseudo-element. ::after accepts the same values for the “content” attribute that we discussed earlier.

Conclusion

The ::before and ::after pseudo-elements allow you to add content to a specific part of an element you have selected in a CSS rule. For instance, the ::before selector could be used to add text before a link. The ::after selector could be used to add an emoji after a paragraph of text.
